安装

对于Airbnb 员工

克隆并初始化monorepo

请运行这些命令：

git clone git@git.musta.ch:airbnb/miniapps.git
cd miniapps
yarn

进入项目

接下来进入你的项目文件夹。

如果您想要开发Airbnb Goji 小程序，请运行 cd packages/airbnb-goji。

如果您想要开发China Checkin小程序，请运行 cd packages/china-checkin。

如果您想要找到所有 Goji 版本的 DLS 组件，请运行 cd packages/dls-demo-goji。

构建并运行

运行 yarn start ，然后用微信开发者工具打开 dist 文件夹。

如果您想要在手机上“预览”您的项目，请运行 yarn build。

如果您想要开发其他平台，请阅读跨平台文档。

对于开源的 Goji 用户

请设置一个 Webpack开发环境，参考 webpack.config.js

[Proposal] 我们稍后可能提供一个 CLI工具 @goji/cli.
